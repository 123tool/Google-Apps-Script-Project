# 👑 XML to JSON

Sistem Konversi XML Enterprise-Grade yang berjalan di atas Google Apps Script.

## 🚀 Fitur Unggulan
- **Advanced XML Parser**: Mendukung rekursi mendalam dan atribut elemen.
- **PDF Engine**: Otomatis membuat dokumen laporan PDF dari hasil konversi.
- **Multi-Channel Alert**: Notifikasi instan ke **Telegram** dan **WhatsApp** sekaligus.
- **Cloud Storage**: Semua hasil konversi tersimpan aman di Google Drive.
- **Modern Dashboard**: Interface gelap futuristik dengan fitur download langsung.

## 🛠 Instalasi
1. Salin `Code.gs` dan `Index.html` ke Google Apps Script.
2. Lengkapi API Keys pada objek `CONFIG`.
3. Berikan izin akses Drive, Telegram, dan WhatsApp Gateway.
4. Deploy sebagai Web App.
