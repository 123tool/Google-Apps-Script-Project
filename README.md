## 📝 Deskripsi
Selamat datang di repositori pusat pengembangan **SPY-E & 123Tool**. Repositori ini berisi koleksi solusi automasi berbasis **Google Apps Script (GAS)** yang dirancang untuk mengubah ekosistem Google Workspace (Sheets, Drive, Forms) menjadi aplikasi web, bot, dan sistem API berstandar industri.

## 📖 Panduan Umum Instalasi (SOP)

Untuk menjalankan seluruh project dalam repositori ini, ikuti langkah-langkah standar berikut:

1.  **Persiapan Editor:**
    * Buka [script.google.com](https://script.google.com).
    * Buat **New Project** di dashboard Apps Script Anda.
2.  **Penyalinan Kode:**
    * Salin isi file `.gs` (Backend) ke bagian script.
    * Jika project memiliki tampilan web, buat file HTML baru (Klik `+` > `HTML`) beri nama sesuai file asli (misal: `Index.html`) dan tempel kodenya.
3.  **Konfigurasi Variabel (`CONFIG`):**
    * Setiap script memiliki objek `const CONFIG`. Pastikan Anda mengisi data yang diperlukan seperti `FOLDER_ID`, `TELEGRAM_TOKEN`, atau `CHAT_ID` sesuai instruksi di dalam komentar kode.
4.  **Aktivasi Trigger (Automasi):**
    * Klik ikon **Triggers (Jam)** di sidebar kiri.
    * Tambahkan trigger baru (misal: `onEdit` untuk sinkronisasi instan atau `Time-driven` untuk bot yang berjalan otomatis).
5.  **Deployment (Web App):**
    * Klik **Deploy > New Deployment**.
    * Pilih jenis: **Web App**.
    * Setel *Execute as:* **Me** dan *Who has access:* **Anyone**.
    * Salin URL Web App yang dihasilkan.
